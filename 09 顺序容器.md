## 顺序容器

| 顺序容器类型    | 含义    |
| --------       | -----:  | 
| vector         | 可变大小数组，连续存储，支持快速随机访问| 
| deque          | 双端队列，支持快速随机访问|
| list           | 双向链表，支持双向顺序访问|     
| forward_list   | 单向链表，支持单向顺序访问|  
| array          | 固定大小数组，支持快速随机访问| 
| string         | 与 vector 相似的容器，但专门用于保存字符；连续存储|
